Cloud Lamp

Cloud lamp es un proyecto con el que se buscaba diseñar un producto de uso común uniendo lo físico con lo digital. Cloud Lamp es una lámpara decorativa que reacciona a la música además de ofrecer una experiencia en realidad aumentada y virtual.
Características: Unity, Vuforia, Arduino
